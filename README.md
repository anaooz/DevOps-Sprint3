# DevOps-Sprint3

### Clone o repositório

```$ git clone https://github.com/anaooz/DevOps-Sprint3.git```

### Acesse a pasta do projeto

```$ cd DevOps-Sprint3```

### Execute seu protejo

No VS Code ou IntelliJ, clique na seta para rodar o projeto; caso não apareça, use o comando ```$ .\mvnw.cmd spring-boot:run```
